# Portfolio Frontend

个人空间前端 - React + TypeScript + TailwindCSS

## 功能页面
- 🏠 Home - 首页
- 👤 Profile - 个人信息
- 📂 Projects - 项目展示
- 📄 Documents - 文档管理
- 🤖 AI Assistant - AI 助手

## 技术栈
- React 18
- TypeScript
- TailwindCSS
- Vite
- Lucide Icons

## 本地开发
```bash
npm install
npm run dev
```

## 构建部署
```bash
npm run build
```
